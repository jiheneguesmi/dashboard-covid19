#  COVID-19 Data Visualization Dashboard

##  Project Overview
This project is an **interactive data visualization dashboard** that provides insights into the global spread and impact of COVID-19.  
It was built using **D3.js**, **HTML**, and **CSS**, with real COVID-19 datasets to visualize confirmed cases, deaths, recoveries, and trends over time and across countries.

The dashboard allows users to explore COVID-19 statistics through **maps, charts, and timelines**, making it easier to understand the evolution of the pandemic.
